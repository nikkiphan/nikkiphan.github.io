Navigation

The navigation bar should be included on all pages with this design system. It includes a menu and different pages of the website.

Example

The navigation bar should be responsive. Adjust the browser to view the different sizes.

  • Home
  • Design
  • About

Accessibility

Follow the color palette and be sure to consider light and dark mode browsers. Also include an alt description for the homepage logo so users know what page they will be directed to.

Usages

Use this navigation bar on all pages in this website.

Code Example

          
    <nav class="navbar navbar-expand-lg navbar-light bg-light sticky-top mr-auto">
      <div class="container-fluid">
        <a class="navbar-brand" href="#">
          <img src="../images/748be7logo.png" alt="logo" width="30" height="30">
        </a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNavDropdown" aria-controls="navbarNavDropdown" aria-expanded="false" aria-label="Toggle navigation">
          <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse justify-content-end" id="navbarNavDropdown">
          <ul class="navbar-nav">
            <li class="nav-item">
              <a class="nav-link active" aria-current="page" href="#">Home</a>
            </li>

            <li class="nav-item">
              <a class="nav-link" href="#">Design</a>
            </li>

          </ul>
        </div>
      </div>
